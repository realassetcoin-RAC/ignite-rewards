How It Works:

  3. Perfect Alignment: Both elements have identical dimensions (40px height, 8px 16px padding)

Code Structure:

<div className="relative">
  <input

  />
  <Button
    variant="outline"
    className="w-full justify-start text-left font-normal h-10 px-4 py-2 pointer-events-none"

    {field.value || '1'}
